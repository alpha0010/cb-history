History

stahta01 2008-09-19 20:29:24

This patch removes the macro wxADJUST_MINSIZE.

This macro is not define when wx26 compatible mode is off.

Current Code::Blocks build defaults to having wx26 compatible mode is on.

But, it can be turn off when wxWidgets 2.8 or 2.9 is compiled.
