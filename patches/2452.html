rhf 2008-04-29 12:00:03

Apparently, I did not include the patch in the initial try. so I'll try again.

This patch speeds up "update.bat" by eliminating the unnecessary copying of unchanged files. The option /u was added to every zip command; the copy commands were changed to xcopy and the /D option was added to every xcopy command.

I have used this for about a month and everything seems to work. Probably similar mods could speed up the 'update' script as well.
