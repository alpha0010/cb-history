All plugins missing when building from source

I have built the x86_64 version of Code::Blocks 10.05svn6285 on Fedora 14 from source using the commands:

./configure --with-contrib-plugins=all --prefix=/spare/local

make

make install

This problem occurs whether or not I run "./bootstrap" before the configure command.

When I start Code::Blocks, I see the following in my log:

Configured 0 tools

Scanning for plugins in /apps/infrafs1/nmusolino/.codeblocks/share/codeblocks/plugins

Loaded 0 plugins

Scanning for plugins in /spare/local/lib64/codeblocks/plugins

Running startup script

::END OF LOG EXCEPT::

When I open the "Manage Plugins" dialog, there are no plugins. As a consequence, I cannot create a new project or file.
