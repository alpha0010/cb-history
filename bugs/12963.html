Open Project with current workspace

I am not sure this is a bug, but it has surprised me several times. I frequently work with a named workspace that includes several projects.

I then want to open a recent project that is not in the workspace. However, selecting the .cbp project from File/Recent projects will ADD the new project to the existing workspace rather than open the project in a separate workspace. This seems odd to me; it seems more like an "Add recent project" option.

Thanks.
