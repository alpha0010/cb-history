Bug #10669 2007-03-18 15:02:17

ondra_holub

file global.h, function cbU2C should return const value

Function is implemented in src/sdk/global.cpp as

// Return multibyte (C string) representation of the string

wxWX2MBbuf cbU2C(const wxString& str)

{

if(platform::unicode)

return str.mb_str(wxConvUTF8);

else

return str.mb_str();

}

So it tries to convert const value to non-const. Function should return const wxWX2MBbuf - then everything seems ok. Without this change I am not able to compile on SUSE Linux 10.1/64bit with gcc 4.1.0.
