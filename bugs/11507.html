File "Not Found" with new project

The following describes what appears to be wrong behavior:

Create a new (console) Project with a main that includes a header file located, say, in C:\Prjs\libs

Project -> Build options... -> Search directories -> Compiler -> Add C:\Prjs\libs

Build and Run successfully.

(1) Open main, right click on include file, select Open "fn"

Get "Not Found:" response.

Close and Save Project

Reopen Project

Repeat (1) and file opens successfully.

This seems incorrect to me. You should not have to close and reopen the Project for (1) to succeed.
