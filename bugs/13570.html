Ok here is a Tutorial for getting Code::Blocks running like it should be

1. Download latest SVN copy

2. Compile and make install code blocks..

3. cd to CodeBlocks/src/plugins/contrib/wxSmith and open the CB project file (wxSmith-unix.cbp).

4. cd CodeBlocks/src/devel -> sudo cp libpropgrid.so /usr/local/share/codeblocks/plugins.
