Text zooms with Ctrl + Caps Lock key combination

Pressing Ctrl + Caps Lock causes the text editor to zoom the text out (make smaller). I can not find a reference to this being a key combination for this function. Because the Tab key is above the Caps Lock, I often hit this instead of Ctrl + Tab (to tab between open files).

This is true with the 8.02 release on Windows and SVN 4966 on Ubuntu.
