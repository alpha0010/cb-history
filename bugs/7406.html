Add a 'what to do' list box on application exit

When having a large project with several files opened and modified, when you exit the application, you receive a message box for each file prompting for what to do (cancel changes or save). Annoying.

It would be better to have a list control with all modified and open files with a check box front of each line. Select all check boxes by default, and add the following button at the bottom of the control : 'Save selected', 'Save all', 'Close all without saving', 'Cancel and return to IDE'...
