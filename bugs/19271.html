#ifdef colouring settings ignored on paste

I have the settings disabled in:

General settings -> C/C++ Editor Settings -> "Interpret #if, #else, #endif to grey out inactive preprocessor code"

For the most part this works correctly (i.e. not greying out any code). However, when I paste existing code containing preprocessor directives into a new file, the preprocessor inactive code is once again greyed out in that file.

Saving and closing and re-opening the file removes the greyed-out effect.

Opening the general settings as above and hitting "ok" without making any changes also removes the greyed-out effect.
