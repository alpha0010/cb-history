Scrollbar theme change with mousewheel

I noticed that when the main window for viewing code has mouse focus, if a user presses and holds on the side of the mousewheel to push it to the left or right to move the horizontal scroll bar the scrollbar changes to a 'basic theme' scrollbar (looks like its from windows 95) and moves respectively with the mousewheel. After the user releases the mousewheel the scrollbar remains in its 'basic theme' until the mouse cursor is moved over the horizontal scrollbar, which then changes back to its regular theme but has no effect on the window (it does not shift the view of the window to the left or right). I have tested this on other windows and they seem to work fine.
