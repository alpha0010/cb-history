duplicated virtual folders

C::B duplicate virtual folders.

To reproduce:

1. Create a virtual folder as a sub-folder of the project.

2. Add some files there.

3. Move the virutal folder by Drag&Drop to another virtual folder. Now the virtual folder is a sub-folder of another virtual folder.

4. Everything looks ok now. Close C::B.

5. Start C::B again.

6. Now you see the moved virtual folder. It is correct. But you see the same folder on its old position (sub-folder of the project) and empty.
