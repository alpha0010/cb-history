Non Printing Underscores

I have tested/reproduced this on 4 separate computers and 2 printers.

If editor is Courier New size 8 (was the default when I installed), and printed with line numbers. The resulting print is actually 7pt without underscores.

Editor with size 7, and without line numbers. Prints 7pt without underscores.

I printed several other sizes, with and without line numbers. The results were when printed without line numbers it is the same pt size as the editor. When printing with line numbers its actually 1pt smaller than what the editor is set at.

I have only found that 7pt doesn't print underscores. In other applications (notepad, openoffice, etc) underscores are printed.
