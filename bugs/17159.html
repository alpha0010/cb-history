File's execute bit not preserved when re-saving file

With Code::Blocks 8.02 on a GNU/Linux system (Fedora 11):

- Create a file and give it the "execute" attribute: chmod a+x somefile

- Load that file in Code::Blocks, modify it and save it.

- The file does not have its execute bit anymore.

Preserving the execute bit would be practical when editing scripts.
