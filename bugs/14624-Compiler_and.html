johnios

Compiler and Debugger toolbars disappear

during (probably at begining) of a debug run, both Compiler and Debugger toolbars disappear from the interface.

These were the only one active with the Main one.

Both were still absent at the end of the run (after switching to the the default layout)

although they are considered active : the view / toolbars checks are active and unchecking / rechecking doesn't help to make them appear again.

I had this bug with a previous version (end august 08) and installed the 5208 build without getting back the toolbars.

(I probably saved the layout)

I had to delete the file(s) in C:\Documents and Settings\MyCount\Application Data\codeblocks to get them back.

I encountered the same problem with the 5208 version. This time, I didn't saved the layouts and got back the toolbars after closing/opening again the software.
