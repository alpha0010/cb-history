phjet 2012-01-24 20:44

2 solutions (on kubuntu)

1/ through console:

setxkbmap -option nsbp=none

2/ through GUI :

system settings -> input devices -> keyboard -> advanced -> "using space key to input non breakable space character" -> select "usual space at any level"
