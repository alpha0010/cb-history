There is a problem with catching DOS stderr by C::B

Workaround:

Create batch file gcc.cmd with following line:

gcc.exe %* 2>&1

and gpp.cmd with:

gpp.exe %* 2>&1

and then change Toolchain executables to gpp.cmd and gcc.cmd
