Missing cbKeyBinder04v19.ini file

Hello,

I've downloaded latest nightly version (20/08/06) Unicode for Windows. I have a problem with assigning shortcuts for saved layouts. When I try to add new shortcut by pressing "Add" button I get message "KeyBinding file corrupted. Please delete it." Unfortunately I didn't found this file. Also I have noticed that I can assign shortcuts to any other "default" option from menu.

On C::B forum "tiwag" told me that I can locate "corrupted" file in "%APPDATA%\codeblocks\cbKeyBinder04v19.ini", but there was no such file. As soon as I have created empty file with that name I was able to add shortcuts for my previously saved layouts.

My conclusion is that C::B was unable to find keybinder file and didn't allow to add shortcuts. I think that it could be easily fixed by creating this file if necessary.

C::B 1.0 revision 2882 wx2.6.3 (Windows, Unicode)

Windows XP sp2 Windows 2000 theme.
