Tab text colour is always black, even if tabs are black

When using a dark OS theme such as Xfce-dusk, tabs appear black or dark grey like most widgets; however, their text colour seems to be hardcoded to black. The result is black text on a black background. This happens with all of the tab styles; with some, it affects only active tabs, others only inactive tabs and some all tabs. Only the default style is usable (active tabs have white text, inactive have a gradient that makes it possible to make out the text).
