Regression: Code::Blocks freezes when opening certain files

With svn 7384, Code::Blocks freezes when opening certain files in the editor. It makes no difference whether it is automatically opened as the result of showing the location of a compiler error or if you open it manually (e.g. from the file manager).

I'm not sure what triggers it exactly, however it seems all files in /usr are affected.

This must have been introduced recently, as it did not happen with previous svn versions.
