Bug #7195 2006-04-20 00:59

andrejv

Keyboard shortcut

Code::Blocks has a 'Ctrl-Alt-G' menu shortcut for "Goto function..." menu entry. This makes it imposible to enter ']' character on keyboards which use 'Alt-Gr-G' to input ']'. This happens on Slovenian keyboard layout and many other Slavic language keyboard layouts.

Slovenian keyboard layout (requires IE):

http://www.microsoft.com/globaldev/keyboards/kbdslove.htm

Other keyboards:

http://www.microsoft.com/globaldev/reference/keyboards.mspx

At least Croatian, Slovak, Serbian (Latin) also use 'Alt-Gr-G' for ']' character.

Andrej
