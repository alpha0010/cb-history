"Switch tabs" behavior change

I wish the "Switch tabs" (CTRL-TAB) keyboard shortcut would automatically select the next tab when you first press it. Or, even better, switch to the last tab that I was on, like Visual Studio does.

Obviously when I press CTRL-TAB I'm trying to switch to another tab, so why have me press it twice? I can imagine someone perhaps just wanting to get a list of the currently open tabs (when many tabs are open) using that shortcut, however it's called "Switch tabs", not "View tabs".

This is really useful when I'm coding something, then need to look at another tab for more information, then switch back, and repeat.
