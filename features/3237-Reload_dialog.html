Reload dialog: move question from title

When a source file opened in the editor is modified outside, a message dialog pops up asking whether to reload it.

Now, the text in the dialog only says that a file has been modified, and presents four options (Yes, No, Reload, Cancel). But the question is missing! Why? Because it is actually part of the title bar of the window: "Reload file?"

Every time that window appears, I spend a few seconds wondering what it is asking me... I think it would be better to move the question in the dialog text!
