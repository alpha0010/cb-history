single file compile without project

we should be able to compile single files which are not part of any project. i know that if some project is open it is hard to actually do this, maybe right click on the tab would give run option which can do that.

reason: the reason to ask for this is that in my college i have noticed ( even when i started c coding ) no one knows about projects and makefiles. they just learn to write a plain c file and run ( probly start with turbo c or gcc ). if codeblocks would support something to kick start basic beginners then i feel it would help the students quite a bit.

maybe the startup can be different / mode of codeblocks each file acts as a project. i would use this while testing some basic c/c++ code. no need to create a new project for each of the tests.
