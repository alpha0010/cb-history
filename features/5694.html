Feature #5694 2013-10-10 15:53

davidallen

Auto synch ProjectManager and open tab

Rather than having to use the editor context menu "Show file in the project tree", I feel it would be nice if this is automatic.

As you open files into the editor and move between tabs, then the projectmanager could automatically highlight the selected file. This is useful when working in files in the same area of the project.

Suggest a checkbox configuration setting in "Settings > Editor > General Settings > Editor Settings > Other Options". Perhaps entitled as "Automatically highlight open file in Project Manager".

As an aside : I noticed this feature since I was accustomed to having it in Netbeans.
