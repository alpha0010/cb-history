support for GNU AS

Hello,

I am sure that adding support for assembly files would be great for all code::blocks community.

For a more personal point of view, I am using Code::blocks with an embedded version of GCC (xilinx microblaze softcore) for a school project. We have to use assembly for bootstrap and interrupt/exceptions code. But I cannot specify where to use "as", so I only use CB to edit "c" files.

It would be great if one could specify how to compile assembly files as well.
