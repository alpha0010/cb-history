Easier access to last viewed document

At the moment, the only way re-view an already opened is:

a) the opened files list - files are listed in order of when first opened

b) the tab bar above the editor - files are listed in order of when first opened.

c) the project management tree.

What would be easier for the user is a system already seen in other IDE's such as MSVS. This is where the last opened document is at the top of a 'documents' pull down menu. As soon as another document is viewed, the last viewed become the 2nd in the list (from the top) and the currently viewed is placed on top. The 2nd in the list becomes the 3rd in the list etc etc ... This way, recently viewed documents are very easy to access, and one does not have to search through a great big list of files to find the file one wants to look at again.
