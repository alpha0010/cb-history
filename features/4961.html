Feature #4961 2010-03-03 19:08

khu

Tab settings be part of chosen language

When using CB for different languages, it may be desirable to have different coding conventions for each language, specifically regarding indentation. When I'm coding in C, I enjoy 4 spaces; when coding in Ruby, I enjoy 2 spaces.

Because the settings under Configure editor > General settings > TAB options are global, this switching between languages (which for me is frequent) is cumbersome.

Would it be possible for these settings (and possibly others?) to be part of the currently chosen language?

Thanks!
