RubyOnRails (Hobo?)

I much prefer using Code::Blocks to either NetBeans or Eclipse for any type of project in any programming language. It's faster and much less difficult to configure. I was wishing that someone might develop another plugin for Code::Blocks that would handle projects in RubyOnRails. Hobo is a plugin for RubyOnRails that adds many more "ease of use" features to Rails. It would be absolutely "perfect" if the same plugin for Code::Blocks would setup and install both Rails and Hobo Project functionality. I'm willing to donate at least $20 per month to the Code::Blocks Project if anyone can help out with this idea.
