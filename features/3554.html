RSS Feeds for Nightlies

It would be nice to have a RSS feed for each nightly build version, that is, a feed for the "generic" Windows version (which is always the first out), and a feed for each of the officially supported build (Ubuntu, FC4, etc.).

Right now, for the Windows build, it's possible to monitor the forum for new posts, but I find myself checking regularly to see if the Ubuntu build is out. Ok, I could build it myself, but I prefer the "official" .deb, with the correct revision number and description and all.

I suppose this could be automatized somehow... Thanks!
