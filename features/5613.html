KeyShortcut: jump to blocking command

I use Ilya Birmans typography keyboard layout. Many typography characters are assigned to AltGr, including English quotes AltGr+K, AltGr+L. AltGr+K works perfectly, but AltGr +L enters some comment.

Settings &#8594; Editor &#8594; Keyboard Shortcuts, press AltGr+L well, theres Line comment assigned to Ctrl+Alt+L (in Windows due to technical reasons AltGr=Ctrl+Alt). But wheres that command? I dug the list for 10 minutes until I found it in DoxyBlocks.

My suggestion. When in Keyboard shortcuts dialog the user clicks Currently assigned field, he jumps to that command.
