default shortcut for "stop debugger"

Maybe it would be a nice idea to add default keyboard shortcut (ctrl + F8?) for "Stop debugger" function.

It is currently possible to assign custom shortcut, but this requires having ""Keyboard shortcuts" plugin active. Having default shortcut will improve debugging workflow for people who (for some reason) keep it disabled.
