Jump-to-line for Tools

In "Tools"->"Configure tools..."->"Edit"->"Launching Options", there should be an option similar to "Launch tool hidden with standard output redirected", except that it would put the output into the "Build messages"/"Build log" section of the "Logs & others" window (or in a new "xxx Tool" tab), so that you can double-click on a line containing a filename/linenumber and jump to that particular line in the file, just like you can do with compiler output.

A lot of tools (ex: Lint-style tools) output filenames/linenumbers of warnings/errors just like a compiler does, and it's a pain to not be able to jump to the offending line just like I can if it were a compiler's output.
