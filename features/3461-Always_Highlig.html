cmb

Always Highlight Missing Braces

There is highlighting of missing braces, parentheses, brackets and so on in Settings > Editor > Syntax Highlighting. That visual display is enabled only when the caret is to the right of the mate-less punctuation. There might be some benefit to highlighting the lonely punctuation at all times.

The default coloring of such highlighting should be changed. Now the foreground color is white which is the same as the default editor background color. If Highlight Line Under Caret is enabled the situation is only a little better. Then the punctuation display is white against light yellow. It is disconcerting for the offending punctuation to disappear as you move the caret around.
