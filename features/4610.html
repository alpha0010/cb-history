Modifier key for mouse wheel scroll page

A nice feature in some tools (i.e. many KDE apps) is using a modifier key to adjust the scroll distance of the mouse wheel.

So, by holding down the shift key, the mouse wheel scrolls a page at a time, rather than the usual several lines. Emacs also offers a mode where holding down a modifier, the scroll reduces to single line steps.

This allows both quick and precise scrolling using only the wheel.
