Feature #4608 2009-05-18 19:25

rjwgnr27

Strip trialing blanks changed lines only

I like having trialing blanks removed from lines I change, but setting the editor setting "Strip trailing blanks", does so on the entire file. This causes large diff files when generating a diff or saving in version control.

A useful compromise is an option to strip trailing blanks only on new and/or changed lines.
