Synchronize between "projects" and edit

Hi all,

There's a useful feature that is in Eclipse for java and that I miss in code::blocks : You have a sort of "synchronize" 2-state button in projects view. When switched on, the file you have focus for editing is automatically selected in projects view. When you have a project that is becoming rather big, it makes finding file in the project tree much faster..
